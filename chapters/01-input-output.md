# 1. 입력과 출력

프로그램은 "입력을 받아서" → "계산해서" → "출력하는" 흐름입니다.
알고리즘 문제도 똑같습니다. 그래서 **입력을 제대로 받는 것**이 첫 관문이에요.

## 출력: `print()`

```python
print("Hello")        # 출력: Hello
print(1 + 2)          # 출력: 3
print("나이:", 20)    # 출력: 나이: 20  (콤마로 나누면 사이에 띄어쓰기가 들어갑니다)
```

## 입력: `input()`

`input()`은 사용자가 키보드로 친 한 줄을 **문자열(글자)로** 받아옵니다.

```python
name = input()        # 사용자가 "철수"라고 입력했다면
print(name)           # 출력: 철수
```

> ▶ **바로 실행해보기** — `input()`은 IDE에서 입력을 기다리며 멈춥니다.
> 아래처럼 입력값을 문자열로 넣으면 복사해서 바로 돌려볼 수 있어요.

```python
name = "철수"         # input() 대신 값을 직접 넣음
print(name)           # 출력: 철수
```

## 중요: 숫자로 받고 싶을 땐 `int()`로 감싸기

`input()`은 무조건 문자열입니다. `"5"`는 글자 5이지 숫자 5가 아니에요.
계산하려면 `int()`(정수)로 바꿔야 합니다.

```python
a = int(input())      # "5"를 입력하면 → 숫자 5
b = int(input())      # "3"을 입력하면 → 숫자 3
print(a + b)          # 출력: 8   (int로 안 바꾸면 "53"이 나옵니다!)
```

> ▶ **바로 실행해보기**

```python
a = int("5")          # int(input()) 대신 값을 직접 넣음
b = int("3")
print(a + b)          # 출력: 8
```

## 핵심: 한 줄에 여러 개 입력받기 — `split()`

코테에서 가장 많이 쓰는 패턴입니다. `"10 20 30"`처럼 한 줄에 여러 값이 들어올 때요.
`split()`은 문자열을 **공백 기준으로 잘라서 리스트로** 만들어줍니다.

```python
# 입력이 "10 20 30" 이라고 가정
data = input().split()      # ['10', '20', '30']  (아직 글자예요!)
print(data)                 # 출력: ['10', '20', '30']
```

> ▶ **바로 실행해보기** — `input()` 자리에 `"10 20 30"` 문자열을 넣으면 똑같이 동작합니다.

```python
data = "10 20 30".split()   # input() 대신 문자열을 직접 split
print(data)                 # 출력: ['10', '20', '30']
```

## 여러 숫자를 한 번에 정수로 — `map(int, ...)`

`map(int, ...)`은 "리스트의 모든 원소에 `int`를 적용해라"는 뜻입니다.
자주 쓰는 마법의 한 줄이에요.

```python
# 입력이 "10 20 30" 이라고 가정
a, b, c = map(int, input().split())
print(a + b + c)            # 출력: 60

# 개수가 정해지지 않았다면 리스트로 통째로 받기
nums = list(map(int, input().split()))
print(nums)                 # 출력: [10, 20, 30]  (이번엔 진짜 숫자!)
```

> ▶ **바로 실행해보기**

```python
a, b, c = map(int, "10 20 30".split())
print(a + b + c)            # 출력: 60

nums = list(map(int, "10 20 30".split()))
print(nums)                 # 출력: [10, 20, 30]  (이번엔 진짜 숫자!)
```

> 💡 거의 모든 코테 문제는 이 세 줄 안에서 입력을 받습니다:
> `n = int(input())` / `a, b = map(int, input().split())` / `arr = list(map(int, input().split()))`
> 이것만 익혀도 절반은 온 겁니다.
> **실전에서는 `input()` 버전을**, 손으로 복붙해 확인할 땐 위 실행본처럼 문자열을 넣어 쓰세요.
