# 4. 조건문 (if)

"만약 ~라면 이걸 하고, 아니면 저걸 해라"를 표현합니다.

## 파이썬의 규칙: 들여쓰기(indent)로 묶는다

`if` 아래 **한 칸 들여쓴 줄들**이 "조건이 참일 때 실행할 코드"입니다.
보통 **스페이스 4칸**으로 들여씁니다. (이게 파이썬의 핵심 문법 규칙!)

```python
score = 85

if score >= 90:
    print("A")        # 이 줄은 실행 안 됨
elif score >= 80:     # elif = "else if" (앞 조건이 거짓일 때 다음 조건 검사)
    print("B")        # 출력: B  ← 여기 실행
else:
    print("C")
```

## 여러 조건 조합

```python
age = 15

if age >= 20:
    print("성인")
elif age >= 14:
    print("청소년")   # 출력: 청소년
else:
    print("어린이")
```

> 💡 들여쓰기를 틀리면 파이썬은 바로 에러를 냅니다. 다른 언어의 `{ }` 대신 들여쓰기가 그 역할을 해요.
> "짝수면 이렇게, 홀수면 저렇게" 같은 분기 처리는 문제풀이의 기본 뼈대입니다.
