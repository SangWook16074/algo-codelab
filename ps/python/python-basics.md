## 1. 입력과 출력

프로그램은 "입력을 받아서" → "계산해서" → "출력하는" 흐름입니다.
알고리즘 문제도 똑같습니다. 그래서 **입력을 제대로 받는 것**이 첫 관문이에요.

### 출력: `print()`

```python
print("Hello")        # 출력: Hello
print(1 + 2)          # 출력: 3
print("나이:", 20)    # 출력: 나이: 20  (콤마로 나누면 사이에 띄어쓰기가 들어갑니다)
```

### 입력: `input()`

`input()`은 사용자가 키보드로 친 한 줄을 **문자열(글자)로** 받아옵니다.

```python
name = input()        # 사용자가 "철수"라고 입력했다면
print(name)           # 출력: 철수
```

### 중요: 숫자로 받고 싶을 땐 `int()`로 감싸기

`input()`은 무조건 문자열입니다. `"5"`는 글자 5이지 숫자 5가 아니에요.
계산하려면 `int()`(정수)로 바꿔야 합니다.

```python
a = int(input())      # "5"를 입력하면 → 숫자 5
b = int(input())      # "3"을 입력하면 → 숫자 3
print(a + b)          # 출력: 8   (int로 안 바꾸면 "53"이 나옵니다!)
```

### 핵심: 한 줄에 여러 개 입력받기 — `split()`

코테에서 가장 많이 쓰는 패턴입니다. `"10 20 30"`처럼 한 줄에 여러 값이 들어올 때요.
`split()`은 문자열을 **공백 기준으로 잘라서 리스트로** 만들어줍니다.

```python
# 입력이 "10 20 30" 이라고 가정
data = input().split()      # ['10', '20', '30']  (아직 글자예요!)
print(data)                 # 출력: ['10', '20', '30']
```

### 여러 숫자를 한 번에 정수로 — `map(int, ...)`

`map(int, ...)`은 "리스트의 모든 원소에 `int`를 적용해라"는 뜻입니다.
자주 쓰는 마법의 한 줄이에요.

```python
# 입력이 "10 20 30" 이라고 가정
a, b, c = map(int, input().split())
print(a + b + c)            # 출력: 60

# 개수가 정해지지 않았다면 리스트로 통째로 받기
nums = list(map(int, input().split()))
print(nums)                 # 출력: [10, 20, 30]  (이번엔 진짜 숫자!)
```

> 💡 거의 모든 코테 문제는 이 세 줄 안에서 입력을 받습니다:
> `n = int(input())` / `a, b = map(int, input().split())` / `arr = list(map(int, input().split()))`
> 이것만 익혀도 절반은 온 겁니다.

## 2. 변수와 자료형

**변수**는 값을 담아두는 상자입니다. `=`로 값을 넣습니다. (수학의 '같다'가 아니라 '담는다'예요.)

```python
age = 20              # 정수 (int)
height = 175.5        # 실수 (float)
name = "영희"         # 문자열 (str)
is_student = True     # 참/거짓 (bool) — True 또는 False
```

### 자료형 확인: `type()`

```python
print(type(20))       # 출력: <class 'int'>
print(type(3.14))     # 출력: <class 'float'>
print(type("hi"))     # 출력: <class 'str'>
print(type(True))     # 출력: <class 'bool'>
```

### 형변환 (타입 바꾸기)

```python
print(int("100"))     # 출력: 100    (글자 → 정수)
print(str(100))       # 출력: 100    (정수 → 글자, 겉보기는 같아도 이제 문자열)
print(float("3.5"))   # 출력: 3.5    (글자 → 실수)
print(int(3.9))       # 출력: 3      (실수 → 정수, 소수점 아래는 그냥 버림)
```

> 💡 입력은 항상 문자열로 들어오니 형변환은 필수입니다.
> "글자 5"와 "숫자 5"는 완전히 다르다는 걸 몸으로 기억하세요.

## 3. 연산자

### 산술 연산자 — `//` 와 `%` 가 핵심

```python
print(7 + 3)          # 출력: 10
print(7 - 3)          # 출력: 4
print(7 * 3)          # 출력: 21
print(7 / 3)          # 출력: 2.3333333333333335   (/ 는 항상 실수 결과)
print(7 // 3)         # 출력: 2     (몫만, 소수점 버림)
print(7 % 3)          # 출력: 1     (나머지)
print(2 ** 10)        # 출력: 1024  (거듭제곱, 2의 10승)
```

### 비교 연산자 — 결과는 True / False

```python
print(5 > 3)          # 출력: True
print(5 == 5)         # 출력: True   (같다는 '=='  두 개! '='는 값 넣기)
print(5 != 3)         # 출력: True   ('!=' 는 다르다)
print(3 >= 3)         # 출력: True
```

### 논리 연산자 — and / or / not

```python
print(True and False) # 출력: False  (둘 다 참이어야 참)
print(True or False)  # 출력: True   (하나만 참이면 참)
print(not True)       # 출력: False  (뒤집기)

x = 7
print(0 < x and x < 10)   # 출력: True   (x가 0보다 크고 10보다 작다)
print(0 < x < 10)         # 출력: True   (파이썬은 이렇게 이어 쓸 수도 있어요!)
```

> 💡 `%`(나머지)는 "짝수/홀수 판별", "배수 판별", "특정 자리 숫자 뽑기"에 끝없이 나옵니다.
> `//`(몫)는 "자릿수 줄이기"에 많이 씁니다. 이 둘은 코테 필수 도구예요.

## 4. 조건문 (if)

"만약 ~라면 이걸 하고, 아니면 저걸 해라"를 표현합니다.

### 파이썬의 규칙: 들여쓰기(indent)로 묶는다

`if` 아래 **한 칸 들여쓴 줄들**이 "조건이 참일 때 실행할 코드"입니다.
보통 **스페이스 4칸**으로 들여씁니다. (이게 파이썬의 핵심 문법 규칙!)

```python
score = 85

if score >= 90:
    print("A")        # 이 줄은 실행 안 됨
elif score >= 80:     # elif = "else if" (앞 조건이 거짓일 때 다음 조건 검사)
    print("B")        # 출력: B  ← 여기 실행
else:
    print("C")
```

### 여러 조건 조합

```python
age = 15

if age >= 20:
    print("성인")
elif age >= 14:
    print("청소년")   # 출력: 청소년
else:
    print("어린이")
```

> 💡 들여쓰기를 틀리면 파이썬은 바로 에러를 냅니다. 다른 언어의 `{ }` 대신 들여쓰기가 그 역할을 해요.
> "짝수면 이렇게, 홀수면 저렇게" 같은 분기 처리는 문제풀이의 기본 뼈대입니다.

## 5. 반복문 (for / while)

같은 작업을 여러 번 반복할 때 씁니다. 알고리즘의 심장이에요.

### `for` + `range()` — 정해진 횟수만큼 반복

`range(n)`은 `0, 1, 2, ..., n-1`을 차례로 만들어줍니다. (끝 숫자 n은 포함 안 함!)

```python
for i in range(5):
    print(i, end=" ")     # 출력: 0 1 2 3 4
# (end=" " 는 줄바꿈 대신 띄어쓰기로 이어 출력하라는 뜻)
```

```python
# range(시작, 끝) : 시작부터 끝-1까지
for i in range(1, 6):
    print(i, end=" ")     # 출력: 1 2 3 4 5

# range(시작, 끝, 간격)
for i in range(0, 10, 2):
    print(i, end=" ")     # 출력: 0 2 4 6 8
```

### 리스트를 직접 순회하기

```python
fruits = ["사과", "바나나", "포도"]
for fruit in fruits:
    print(fruit)
# 출력:
# 사과
# 바나나
# 포도
```

### `while` — 조건이 참인 동안 반복

```python
n = 5
while n > 0:
    print(n, end=" ")     # 출력: 5 4 3 2 1
    n = n - 1             # n을 1씩 줄임 (이게 없으면 무한 반복!)
```

### `break` (탈출) 와 `continue` (건너뛰기)

```python
for i in range(10):
    if i == 5:
        break             # i가 5가 되면 반복 완전 종료
    print(i, end=" ")     # 출력: 0 1 2 3 4

for i in range(5):
    if i == 2:
        continue          # i가 2일 때만 건너뛰고 다음으로
    print(i, end=" ")     # 출력: 0 1 3 4
```

> 💡 "N번 반복", "리스트 전부 확인", "조건 만족할 때까지" — 문제의 90%가 반복문 안에서 벌어집니다.
> `range`의 끝 숫자가 포함 안 된다는 점을 꼭 기억하세요. (초보자 단골 실수)

## 6. 리스트 (list)

여러 값을 **순서대로** 담는 상자입니다. 알고리즘에서 가장 많이 쓰는 자료형이에요.

### 생성과 인덱싱

```python
arr = [10, 20, 30, 40, 50]

print(arr[0])         # 출력: 10   (첫 번째, 번호는 0부터!)
print(arr[2])         # 출력: 30
print(arr[-1])        # 출력: 50   (-1은 맨 뒤)
print(len(arr))       # 출력: 5    (길이/개수)
```

### 슬라이싱 — 일부만 잘라내기 `[시작:끝]`

```python
arr = [10, 20, 30, 40, 50]

print(arr[1:4])       # 출력: [20, 30, 40]   (1번부터 4번 앞까지 = 1,2,3번)
print(arr[:3])        # 출력: [10, 20, 30]   (처음부터 3번 앞까지)
print(arr[2:])        # 출력: [30, 40, 50]   (2번부터 끝까지)
```

### 값 추가/변경/삭제

```python
arr = [10, 20, 30]

arr.append(40)        # 맨 뒤에 추가
print(arr)            # 출력: [10, 20, 30, 40]

arr[0] = 99           # 값 변경
print(arr)            # 출력: [99, 20, 30, 40]

arr.pop()             # 맨 뒤 값 꺼내서 삭제
print(arr)            # 출력: [99, 20, 30]
```

### 정렬과 뒤집기

```python
arr = [3, 1, 4, 1, 5, 9, 2]

arr.sort()            # 오름차순 정렬 (원본이 바뀜)
print(arr)            # 출력: [1, 1, 2, 3, 4, 5, 9]

arr.sort(reverse=True)  # 내림차순
print(arr)            # 출력: [9, 5, 4, 3, 2, 1, 1]
```

### 2차원 리스트 (표/격자)

```python
grid = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9],
]

print(grid[0])        # 출력: [1, 2, 3]   (첫 번째 줄)
print(grid[1][2])     # 출력: 6           (1번 줄의 2번 칸)

# 모든 칸 순회
for row in grid:
    for value in row:
        print(value, end=" ")
# 출력: 1 2 3 4 5 6 7 8 9
```

> 💡 리스트는 사실상 모든 문제에서 등장합니다. `append`, `sort`, 인덱싱, 슬라이싱은 반사적으로 나올 만큼 익히세요.
> 2차원 리스트는 지도/미로/격자 문제(BFS/DFS)의 기본입니다.

## 7. 문자열 (string)

문자열도 리스트처럼 **글자들이 순서대로** 있는 형태라, 인덱싱/슬라이싱이 똑같이 됩니다.

### 인덱싱과 슬라이싱

```python
s = "algorithm"

print(s[0])           # 출력: a
print(s[-1])          # 출력: m
print(s[0:4])         # 출력: algo
print(len(s))         # 출력: 9
```

### 자주 쓰는 메서드

```python
s = "hello world"

print(s.upper())          # 출력: HELLO WORLD  (대문자)
print(s.lower())          # 출력: hello world  (소문자)
print(s.replace("o", "0"))# 출력: hell0 w0rld  (바꾸기)
print(s.split())          # 출력: ['hello', 'world']  (공백으로 자르기)
print(s.count("l"))       # 출력: 3   (l의 개수)
```

### `join` — 리스트를 문자열로 합치기 (split의 반대)

```python
words = ["a", "b", "c"]
print("".join(words))     # 출력: abc
print("-".join(words))    # 출력: a-b-c   (사이에 - 를 끼워서 합침)
```

### 문자 ↔ 숫자(아스키 코드) — `ord` / `chr`

문자를 숫자로 다뤄야 하는 문제(암호화, 알파벳 순서 계산 등)에서 씁니다.

```python
print(ord("A"))       # 출력: 65    (문자 → 숫자)
print(ord("a"))       # 출력: 97
print(chr(65))        # 출력: A     (숫자 → 문자)
print(chr(ord("A") + 1))  # 출력: B  (A 다음 글자)
```

> 💡 문자열 뒤집기 `s[::-1]`, 대소문자 변환, 특정 문자 세기, 알파벳 순서 계산은 문자열 문제 단골입니다.
> `join`과 `split`은 짝으로 기억하세요. (자르기 ↔ 합치기)

## 8. 딕셔너리와 셋 (dict / set)

### 딕셔너리 (dict) — 이름표(key)로 값을 찾기

리스트는 번호(0,1,2...)로 찾지만, 딕셔너리는 **key(열쇠)**로 값을 찾습니다.

```python
scores = {"철수": 90, "영희": 85, "민수": 70}

print(scores["영희"])         # 출력: 85
scores["철수"] = 100          # 값 변경
scores["지수"] = 95           # 새 항목 추가
print(scores)                 # 출력: {'철수': 100, '영희': 85, '민수': 70, '지수': 95}
```

### `get` — 없는 key를 안전하게 조회

```python
scores = {"철수": 90}

# print(scores["영희"])  ← 없는 key라 에러 발생!
print(scores.get("영희"))         # 출력: None  (에러 대신 None)
print(scores.get("영희", 0))      # 출력: 0     (없으면 0으로 (기본값 지정))
```

### 개수 세기 패턴 (아주 자주 나옴)

```python
text = "banana"
count = {}
for ch in text:
    count[ch] = count.get(ch, 0) + 1   # 있으면 +1, 없으면 0에서 시작
print(count)              # 출력: {'b': 1, 'a': 3, 'n': 2}
```

### 셋 (set) — 중복 없는 모음

순서가 없고, **같은 값은 하나만** 담깁니다. "중복 제거"와 "빠른 포함 검사"에 씁니다.

```python
nums = [1, 2, 2, 3, 3, 3, 4]
unique = set(nums)
print(unique)             # 출력: {1, 2, 3, 4}   (중복이 사라짐)
print(len(unique))        # 출력: 4              (종류가 몇 개?)

print(3 in unique)        # 출력: True   (들어있는지 검사 — 아주 빠름)
```

> 💡 딕셔너리는 "무엇이 몇 번 나왔나", "이름으로 정보 찾기"에 필수입니다.
> 셋은 "중복 제거"와 "이미 방문했나?" 같은 검사에 씁니다. `in` 검색이 리스트보다 훨씬 빨라요.

## 9. 함수 (def)

반복되는 작업에 **이름을 붙여** 재사용하는 것입니다. `def`로 만듭니다.

```python
def add(a, b):        # a, b는 받을 값 (매개변수)
    return a + b      # return = 결과를 돌려줌

result = add(3, 5)
print(result)         # 출력: 8
```

### 매개변수와 반환값

```python
def is_even(n):           # 짝수인지 판별하는 함수
    if n % 2 == 0:
        return True
    else:
        return False

print(is_even(4))         # 출력: True
print(is_even(7))         # 출력: False
```

### 기본값 인자 — 안 넘기면 기본값 사용

```python
def greet(name, greeting="안녕"):
    return greeting + ", " + name

print(greet("철수"))              # 출력: 안녕, 철수      (greeting 생략)
print(greet("영희", "반가워"))    # 출력: 반가워, 영희    (직접 지정)
```

> 💡 문제를 작은 조각으로 나눠 풀 때 함수를 씁니다.
> 프로그래머스는 대부분 `def solution(...):` 안에 답을 채우는 형태라, 함수 정의/반환은 꼭 익혀야 합니다.

## 10. 자주 쓰는 내장 함수

파이썬이 기본 제공하는, 코테에서 손이 자주 가는 함수들입니다.

### 합/최대/최소/정렬/절댓값

```python
arr = [3, 1, 4, 1, 5]

print(sum(arr))           # 출력: 14   (전체 합)
print(max(arr))           # 출력: 5    (최댓값)
print(min(arr))           # 출력: 1    (최솟값)
print(sorted(arr))        # 출력: [1, 1, 3, 4, 5]  (정렬된 새 리스트 반환, 원본은 그대로)
print(abs(-7))            # 출력: 7    (절댓값)
```

### `sorted`로 내림차순 / 기준 정렬

```python
arr = [3, 1, 4, 1, 5]
print(sorted(arr, reverse=True))     # 출력: [5, 4, 3, 1, 1]

words = ["banana", "kiwi", "apple"]
print(sorted(words, key=len))        # 출력: ['kiwi', 'apple', 'banana']  (길이순)
```

### `enumerate` — 번호와 값을 함께

```python
fruits = ["사과", "바나나", "포도"]
for i, fruit in enumerate(fruits):
    print(i, fruit)
# 출력:
# 0 사과
# 1 바나나
# 2 포도
```

### `zip` — 두 리스트를 짝지어서

```python
names = ["철수", "영희"]
ages = [20, 22]
for name, age in zip(names, ages):
    print(name, age)
# 출력:
# 철수 20
# 영희 22
```

> 💡 `sum/max/min/sorted`는 직접 반복문 짜지 않고 한 줄로 끝내줍니다.
> `enumerate`(번호 필요할 때), `zip`(여러 리스트 동시에)은 코드를 훨씬 깔끔하게 만들어줍니다.

## 11. 리스트 컴프리헨션

리스트를 **한 줄로** 만드는 파이썬만의 문법입니다. 처음엔 낯설지만 익히면 정말 편해요.

### 기본형: `[식 for 변수 in 반복대상]`

```python
# 일반 방식
squares = []
for i in range(5):
    squares.append(i * i)
print(squares)            # 출력: [0, 1, 4, 9, 16]

# 컴프리헨션 (위와 완전히 같은 결과, 한 줄로)
squares = [i * i for i in range(5)]
print(squares)            # 출력: [0, 1, 4, 9, 16]
```

### 조건 붙이기: `[식 for 변수 in 반복대상 if 조건]`

```python
# 0~9 중 짝수만
evens = [i for i in range(10) if i % 2 == 0]
print(evens)              # 출력: [0, 2, 4, 6, 8]
```

### 입력받을 때 자주 쓰는 형태

```python
# 입력이 "3 1 4 1 5" 라고 가정 — 각 숫자를 제곱한 리스트
nums = [int(x) ** 2 for x in input().split()]
# 입력 "3 1 4 1 5" → nums = [9, 1, 16, 1, 25]
```

> 💡 컴프리헨션은 "리스트를 가공해 새 리스트 만들기"를 짧게 해줍니다.
> 몰라도 반복문으로 다 되지만, 익혀두면 코드가 짧아지고 읽기 쉬워집니다. 코테 답안에서 정말 많이 보게 될 거예요.

## 마무리 — 이제 뭘 해야 하나요?

여기까지 손으로 쳐봤다면 문제풀이의 **문법 기본기**는 갖춰졌습니다.
이제 프로그래머스에서 다음 순서로 풀어보세요.

1. **Lv.0 / 입문** — 위 문법을 그대로 쓰는 문제들. 입력받기·조건·반복·리스트 위주.
2. **Lv.1** — 문자열, 딕셔너리, 정렬을 섞어 쓰는 문제. 이 자료의 6~11장이 바로 나옵니다.

**막힐 때 체크리스트**
- 입력을 숫자로 안 바꿨나? (`int`, `map`)
- `range` 끝 숫자가 포함 안 되는 걸 놓쳤나?
- 리스트 번호가 0부터인 걸 깜빡했나?
- 들여쓰기가 맞나?

문법은 외우는 게 아니라 **손에 붙이는** 겁니다. 이 자료를 2~3번 반복해서 쳐보세요. 화이팅! 🚀
